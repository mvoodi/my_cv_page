Projects

SeaBattle

A console-based implementation of the classic "Battleship" game. This project is a text-based version of the game "Battleship" written in Java. The program creates a 7x7 grid for placing ships of various lengths (single-deck, double-deck, and triple-deck), allows player interaction through the console, enables input of shot coordinates, and tracks hits, misses, and sunk ships. The game supports scoring, player leaderboard sorting, and offers the option to replay.

Wheel of Fortune

An interactive game inspired by the TV show. This Java program is a console-based word-guessing game that combines elements of team play and competition. Players take turns guessing letters to uncover a hidden word from a shuffled list. During setup, players are registered, and their turn order is randomized. Each hidden word includes a description to provide hints. Players earn points for correctly guessing letters, and they can attempt to guess the entire word to win instantly. If a player reaches the maximum possible score or guesses the word correctly, they are declared the winner. The game also includes features like tracking previously named letters, handling invalid guesses, and offering a final chance for all players to guess the word if one player reaches the maximum score. Various interactive and visual enhancements, such as color-coded text and real-time updates, make the gameplay engaging.

Store Management

This Java program is a console-based application designed for managing a company with different roles, such as a director, seller, accountant, and warehouse manager. It demonstrates object-oriented programming (OOP) principles and interacts with a database to manage various operations like adding/removing employees, selling products, updating product prices, and more. The system requires users to log in using predefined credentials for each role. After logging in, they are presented with role-specific menus, which provide functionality for tasks like viewing the company budget, changing salaries, accepting or shipping products, and managing product details. The program is structured with multiple classes and methods that handle the respective responsibilities of each role, supporting efficient employee and product management. The use of OOP allows for modularity and scalability, making it easy to add or modify features as needed.
